# HR Management API

A RESTful API built with ASP.NET Core 8

## Tech Stack
- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- JWT Authentication
- Swagger

## Features
- JWT Authentication (Register/Login)
- Department Management with Employee Count
- Product & Category Management
- Employee Management

## API Endpoints
- POST /api/Account/Register
- POST /api/Account/Login
- GET/POST /api/Category
- GET/POST/PUT /api/Department
- GET/POST/PUT /api/Product
- GET /api/Employee/{id}
